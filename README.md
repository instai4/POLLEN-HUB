# PollenLab

A visually rich immersive landing page inspired by bees, pollination, and the beauty of nature. The experience combines 3D visuals, smooth scrolling, animated particles, and ambient audio to create a memorable interactive story

## Overview

This project is a single-page website built to showcase a premium, artistic experience centered around pollinators and the natural world. It includes:

- A fullscreen 3D bee experience using model-viewer
- Smooth scrolling and section-based motion
- Scroll reveal animations and interactive card effects
- Floating particle effects and ambient sound synthesis
- A custom-themed UI with bee-inspired visuals

## Technologies Used

- HTML5 for structure and content
- CSS3 for layout, animations, responsive styling, and theme design
- JavaScript for interactivity, scroll effects, particle animation, and audio
- Google Fonts for typography
- Font Awesome for icons
- Model Viewer for rendering the 3D bee model
- Lenis for smooth scroll behavior
- Canvas API for animated pollen particles
- Web Audio API for ambient nature-like sound generation

## Project Structure

- index.html — page layout and sections
- style.css — visual styling, animations, and responsive design
- script.js — scroll logic, reveal effects, particles, audio, and interactivity
- bee_flying.glb — 3D bee model
- bee1.png / bee3.png — gallery images

## How to Run

Open index.html in your browser, or serve the folder using a simple local server if preferred.

Example:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Notes

The project is fully static and does not require any package installation.
